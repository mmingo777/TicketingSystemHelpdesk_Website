💡 What MetroDesk Can Do

Submit a Ticket

Employees can quickly report issues by filling out a simple form with the title, description, priority, and optional file attachment.

Track & Resolve

IT staff and admins can view ticket details, assign agents, leave resolution comments, and update ticket statuses in real time.

View Reports

Gain visual insight through pie charts and summaries of ticket statuses. Track trends over time to improve efficiency.

🔒 Designed for Real Teams

MetroDesk isn’t just a classroom prototype — it’s built with real-world deployment in mind. The system uses a live SQL Server hosted in SQL Server Management Studio, where all tickets, users, and comments are stored securely.

Each staff member uses a secure login tied to their official @nytransit.net email, ensuring that only authorized employees can submit or manage tickets. This prevents unauthorized access and keeps internal data protected.

  • 🔐 Role-based access for Admins, IT Staff, and Employees
  • 📂 Attachments, comments, history tracking, and status updates
  • 📊 Live dashboard reports and ticket summaries
